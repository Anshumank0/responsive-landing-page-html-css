# Responsive landing page website

## Description
A modern and fully responsive landing page website built using pure HTML and CSS.

## Tech Stack
- HTML5
- CSS3
- Flexbox
- Media Queries
- Google Fonts (Poppins)

## Features
- Fully responsive layout
- Modern product showcase design
- Hover scale effect
- Structured layout using Flexbox

## Responsiveness
- Desktop
- Tablet
- Mobile devices
